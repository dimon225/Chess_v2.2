The Game starts by loading all needed classes

This SplashScreen is shown for a moment after all classes are loaded

the game starts:

 

The Game can be played locally or online by default the game is local but if the user decides to play online he simply needs to go to the options menu

click either connect As Server or connect aS Client the game is by default played on port 9999. However the client side of online game would need to ask the user for the server's ip address the client enters it and clicks ok that will connect him automatically to the server who's is also a client but he waits for connection. As you can see the options menu offers some more choices

You can flip the board back and forth, which will simply changesthe view the data or positions are not changed when you flip the board there are only two states normal board or flipped board:

NORMAL_BOARD: has white pieces at the bottom

FLIPPED_BOARD: has white pieces at the top

Choose_Icon simply pops up a JDialog that has icons so the user can choose the icon he likes to represent himself in the game

Set Player Names will do the same but for the names the user will choose the names to represent himself in the game

If the game is played online both name and path to the icon is sent to the other player through a socket  sohe would also see the changes the player that chose the icon or name made.

 

The help menu offers only two choices:

About: If you click about menu the JDailog will pop up and show the version and who made the game

Chess Rules: If you click chess rules menu the JDailog will pop up and show the rules of the chess game.

User Manual: if you click on the user manual you will the JEditorPane

will pop up with the user manual like this one

 

The File menu however offers the most choices to the user:

New Game: simply restarts the game by disposing of the view and creating a new one.

Save Player: the save player simply saves the array list of players

Load Player: simply loads the array list of players

Save Game: simply saves the game and all captured pieces into two different files

Load Game: simply loads the game and captured pieces if any

Save as..:  simply saves the game and all captured pieces into two different files to the file of the users choice the JFileChooser will popup to ask where the user wants the game to be saved

Load from file: simply loads from the selected file the JFileChooser pops up

asks the user for the file and loads from that file;
